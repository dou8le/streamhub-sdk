A Readable Stream that emits Content for a Livefyre Collection as sourced from StreamHub's Bootstrap APIs. This Stream emits Content in descending order by bootstrap page
Name Type Description
opts object Configuration options
Name Type Default Description
network string The StreamHub Network of the Collection
siteId string The StreamHub Site ID of the Collection
articleId string The StreamHub Aritcle ID of the Collection
environment string optional If not production, the hostname of the StreamHub environment the Collection resides on
bootstrapClient LivefyreBootstrapClient optional A Client object that can request StreamHub's Bootstrap web service
replies boolean false optional Whether to read out reply Content
comparator optional Indicate the order in which you'd like to read the archive. Default is CollectionArchive.comparators.CREATED_AT_DESCENDING (newest first). You can also pass other values of CollectionArchive.comparators
